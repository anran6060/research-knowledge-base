# 美伊双方死磕下的资产逻辑思考

一、上周家事缠身，所幸群内未负所托

这几天家人住院，忙完群内策略后，公众号实在无力更新。所幸，再忙也没有耽误正事--群内本轮完美准确把握了债市回调的空头波段，且持续提醒不要轻易抄底。

当前债市的核心逻辑已属实切换：外部因素才是定价主引擎，避险交易反而退居次席，至于国内数据、资金面、股债跷跷板这些，更是沦为无人问津的边角料。没人交易了。

二、刚刚过去的十几个小时里。①、懂王出手，企图拿捏伊朗能源命脉
![](images/img_001.png)美军对哈尔克岛发动"猛烈空袭"，将该岛所有军事目标夷为平地。这座岛屿承担着伊朗约90%的原油出口（约155万桶/日），是德黑兰的经济生命线。懂王随后高调宣布：石油基础设施完好无损--言下之意，"我手握开关，随时可以让你的钱包归零"。
②、伊朗反击，直戳霸权颜面。![](images/img_002.png)伊朗则把美国驻伊拉克大使馆给炸了。伊朗方面称摧毁了该大使馆的防空系统。③、两种逻辑碰撞得火花四溅。
![](images/img_003.png)一边是民族存亡倒逼出的神教血性--制裁绞杀、军事威慑、经济窒息，退无可退，唯有以攻为守。![](images/img_004.png)另一边是"极限施压"失效后的恼羞成怒--反复变脸、威胁加码，却发现对手不再吃TACO这一套，于是只能继续挥舞大棒，试图用更猛烈的袭击找回场子。三、双方这轮死磕下的逻辑简述。①、原油：霍尔木兹海峡相当于断航，供给忧虑加大，释放的储备的消息对原油价格来说只能算釜底抽薪式的短暂降温，在双方无法坐上谈判桌之前，原油易涨难跌的主逻辑难以变化。②、美元：原油价格强关联，原油走强，美元自然强势，除非大家不用美元来结算原油。风险资产承压，流动性因素像一把悬而未决的利刃，随时可能引爆美元走强③、黄金：市场交易流动性因素，资金预防性抛压，形成震荡偏弱走势。但基于全球长期对去美元化+债务货币化+地缘风险的综合考虑后，在杀流动性阶段完成后，黄金依然是具备长期配置价值的最佳资产。④、债市则相对复杂，但当前阶段利空明确。

1）利率债短线逻辑：流动性危机+滞胀预期。

![](images/img_005.png)关于我们的利率债，流动性危机尚存+滞胀预期共同形成了短线主逻辑，该逻辑下利率债会偏空运行，毕竟全球债市都在交易这个叙事，我们很难独善其身。

2）懂王两种剧本，殊途同归。

![](images/img_006.png)TACO。懂王找个台阶下，姿态性妥协，中国国运强化，东升西落逻辑加速 → 利空债市。![](images/img_007.png)不TACO。被伊朗神权抵抗拖入泥潭，同样强化中国国运，东升西落逻辑加速 → 利空债市。

![](images/img_008.png)两条路看似相反，终点一致：战略定力对比下，天平向中国倾斜，风险资产重估（这也是中国股市这几天强于其他国家的重要原因），债市承压。

3）唯一的多头路径：衰退倒逼宽松

![](images/img_009.png)需同时满足以下条件链：战争拖累美国AI叙事+经济叙事 → 最大经济体确认衰退 → 中国进出口受拖累 → 全球衰退叙事强化 → 被迫大幅降准降息 → 利好债市。

![](images/img_010.png)这条路径依赖层层传导，且终端是"被迫"二字--意味着需以经济失速为代价，空间和时间都存在极大不确定性，何况有东升西落逻辑在。因此当前市场尚未定价这一极端情形。
