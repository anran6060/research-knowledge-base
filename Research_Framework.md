## 一、投资理念与原则

### 1.1 核心投资理念

**价值投资为基，趋势博弈为辅**
- 长期持有：基于基本面深度研究，寻找护城河深厚的标的
- 中期波段：把握行业周期与政策拐点，赚取β收益
- 短期交易：利用市场情绪与事件驱动，获取α超额

### 1.2 决策原则

| 原则 | 说明 |
|-----|------|
| **能力圈原则** | 只投自己能看懂的行业与公司 |
| **安全边际** | 买入价格必须有足够的下行保护 |
| **逆向思维** | 人弃我取，人取我予 |
| **概率思维** | 追求高胜率+合理赔率的组合 |
| **止损纪律** | 错了就认，及时止损 |

### 1.3 资产配置理念

```
核心仓位 (60-70%)：高确定性标的，长期持有
卫星仓位 (20-30%)：主题轮动，中期波段
机动仓位 (10-20%)：事件驱动，短期交易
```

## 二、股票投研框架

### 2.1 自上而下的分析框架

**第一层：宏观择时**
- 经济周期判断（复苏/过热/滞胀/衰退）
- 货币政策取向（宽松/中性/紧缩）
- 财政政策力度（扩张/中性/收缩）
- 股债性价比（ERP分位数）

**第二层：行业比较**
- 行业景气度（营收增速、利润增速）
- 政策导向（支持/中性/限制）
- 供需格局（产能利用率、库存周期）
- 估值水平（PE/PB历史分位）

**第三层：个股精选**
- 商业模式（护城河分析）
- 财务质量（ROE、现金流、负债率）
- 估值水平（绝对估值+相对估值）
- 催化剂（业绩拐点、订单落地、政策催化）

### 2.2 个股分析模板

| 维度 | 分析要点 | 关键指标 |
|-----|---------|---------|
| **商业模式** | 怎么赚钱？护城河是什么？ | 毛利率、市占率、客户粘性 |
| **成长空间** | 行业空间？公司增速？ | 营收CAGR、TAM、渗透率 |
| **财务质量** | 盈利能力？现金流？杠杆？ | ROE、经营现金流/净利润、资产负债率 |
| **估值水平** | 贵还是便宜？ | PE/PB/PS分位数、DCF估值 |
| **管理层** | 靠谱吗？ | 历史兑现率、股权结构、激励计划 |
| **风险因素** | 最大风险是什么？ | 政策风险、竞争格局、技术替代 |

### 2.3 持仓标的监控清单

**日常监控（每日）**:
- [ ] 股价异常波动（>±5%）
- [ ] 成交量异常（>3倍均量）
- [ ] 重大公告（业绩预告、订单、政策）
- [ ] 行业新闻（政策、竞争格局变化）
- [ ] 北向资金/主力资金流向

**定期复盘（每周）**:
- [ ] 持仓标的基本面变化
- [ ] 估值水平变化（PE/PB分位）
- [ ] 行业景气度跟踪
- [ ] 技术面关键点位（支撑/压力）

## 三、债券投研框架

### 3.1 利率分析框架

**基本面驱动**:
- 经济增长（GDP、PMI、工业增加值）
- 通货膨胀（CPI、PPI、通胀预期）
- 就业数据（失业率、非农就业）

**政策面驱动**:
- 货币政策（基准利率、公开市场操作）
- 财政政策（赤字率、专项债发行）
- 监管政策（MPA考核、资管新规）

**资金面驱动**:
- 银行间流动性（DR007、R007）
- 央行操作（MLF、逆回购、降准）
- 外汇占款、财政存款

### 3.2 信用分析框架

**主体信用评估**:
| 维度 | 指标 | 权重 |
|-----|------|------|
| 盈利能力 | ROA、EBITDA利润率 | 20% |
| 现金流 | 经营现金流/总负债 | 25% |
| 杠杆水平 | 资产负债率、净负债率 | 25% |
| 偿债能力 | 利息保障倍数、流动比率 | 20% |
| 外部支持 | 股东背景、政府支持 | 10% |

**行业信用排序**（从优到劣）:
1. 利率债（国债、政金债）- 无信用风险
2. 高等级信用债（AAA央企、地方国企）
3. 中等级信用债（AA+优质民企）
4. 低等级信用债（AA及以下）- 谨慎参与

### 3.3 国债期货策略

**核心指标监控**:
- 基差（期货价格-现货价格）
- IRR（隐含回购利率）
- 期限利差（10Y-2Y、10Y-5Y）
- 持仓量/成交量变化

**技术面分析**:
- 关键价位（支撑/阻力）
- 均线排列（MA5/MA10/MA20）
- MACD、RSI、KDJ指标
- 形态识别（头肩顶/底、双顶/底）

## 四、转债投研框架

### 4.1 基础认知

**可转债本质**: 可转换为发行公司股票的债券，兼具债性和股性

**核心条款**:
| 条款 | 说明 | 触发条件 |
|-----|------|---------|
| 转股权 | 按约定价格转为股票 | 发行后6个月至到期日 |
| 赎回条款 | 公司有权按面值+利息赎回 | 股价≥转股价130%持续15天 |
| 回售条款 | 持有人有权按面值回售 | 股价<转股价70%持续30天 |
| 下修条款 | 董事会有权提议下修转股价 | 股价<转股价85%持续10天 |

### 4.2 核心估值指标

| 指标 | 公式 | 参考区间 |
|-----|------|---------|
| 转股价值 | 100/转股价×正股价 | - |
| 转股溢价率 | (转债价-转股价值)/转股价值×100% | 0-50%（平衡型） |
| 纯债价值 | 未来现金流折现 | 85-95元 |
| 到期收益率(YTM) | 持有到期的年化收益 | >0为债性保护 |

### 4.3 四大策略体系

#### 策略一：双低策略（经典防御）

**双低值** = 转债价格 + 转股溢价率×100

**筛选条件**:
- 双低值 < 130
- 剩余期限 > 1年
- 评级 ≥ AA-
- 正股市值 > 50亿
- 日均成交额 > 5000万

**适用场景**: 震荡市、熊市防御

#### 策略二：低溢价进攻策略

**筛选条件**:
- 转股溢价率 < 15%
- 转债价格 < 130元（避免强赎风险）
- 正股20日涨幅 > 10%（趋势确认）
- 正股波动率 > 40%（期权价值）
- 行业景气度向上

**适用场景**: 牛市进攻

#### 策略三：下修博弈策略

**筛选条件**:
- 股价/转股价 < 85%（接近触发线）
- 剩余期限 < 3年（时间压力）
- 大股东持债比例 > 30%（下修动力）
- 回售期临近 < 1年（回售压力）
- 财务费用/净利润 > 50%（利息压力）

**博弈时点**: 触发前介入，股东大会通过后卖出

**适用场景**: 独立条款博弈，穿越牛熊

#### 策略四：高YTM防御策略

**筛选条件**:
- YTM > 3%
- 评级 ≥ AA
- 剩余期限 < 5年
- 正股股息率 > 4%
- 有息负债/EBITDA < 5

**适用场景**: 熊市防御、类债替代

### 4.4 银行转债专项策略

**银行转债评分模型**（满分100分）:

| 指标 | 分值 | 标准 |
|-----|------|------|
| 正股股息率 > 5% | 20分 | 高股息防御 |
| 正股PB < 0.8倍 | 15分 | 估值安全 |
| 转债YTM > 2.5% | 15分 | 债底保护 |
| 转股溢价率 < 20% | 15分 | 股性价值 |
| 评级AAA/AA+ | 10分 | 信用安全 |
| 日均成交 > 1亿 | 10分 | 流动性 |
| 剩余期限 > 2年 | 5分 | 久期可控 |
| 无大额解禁 | 5分 | 股价压力 |
| 区域经济基础好 | 5分 | 城商行适用 |

**评分应用**:
- ≥80分：重仓买入
- 60-80分：标准配置
- <60分：回避

**2024H1经典案例**:
- 中信转债：2月110元→6月128元，+16%（低溢价进攻）
- 兴业转债：1月105元→6月118元，+12%（高YTM防御）
- 紫银转债：3月103元→4月115元，+11.7%（下修博弈）

## 五、商品与衍生品

### 5.1 商品分析框架

**分析维度**:
- 宏观需求（全球经济增长、中国需求）
- 供给约束（产能、库存、地缘风险）
- 金融属性（美元走势、通胀预期）
- 季节性因素（农产品、能源）

### 5.2 衍生品策略

**期权策略**:
- 方向性策略：买入看涨/看跌期权
- 波动率策略：跨式、宽跨式
- 收入策略：卖出备兑、卖出跨式
- 对冲策略：保护性看跌、领口策略

**期货策略**:
- 趋势跟踪：均线突破、动量策略
- 套利策略：期现套利、跨期套利、跨品种套利
- 对冲策略：套期保值、Beta对冲

## 六、风险管理框架

### 6.1 个券风险管理

**风险清单**:
| 风险类型 | 识别信号 | 应对措施 |
|---------|---------|---------|
| 估值风险 | PE/PB历史高位 | 减仓或止盈 |
| 业绩风险 | 业绩miss、指引下调 | 及时止损 |
| 政策风险 | 监管政策变化 | 评估影响，必要时退出 |
| 流动性风险 | 日均成交萎缩 | 控制仓位 |
| 信用风险 | 评级下调、债务逾期 | 回避或止损 |

### 6.2 组合风险管理

**仓位管理**:
- 单券仓位 ≤ 5%（股票）/ ≤ 10%（债券）
- 行业集中度 ≤ 20%
- 股债配比：根据ERP分位数动态调整

**止损纪律**:
| 资产类别 | 止损线 | 减仓线 |
|---------|-------|-------|
| 股票 | -8% | -15%减仓50% |
| 转债 | -5% | -10%减仓50% |
| 债券 | -3% | -5%减仓50% |
| 组合整体 | - | 回撤>15%减仓至50% |

### 6.3 市场风险管理

**情绪指标监控**:
- 破面转债数量（>30只为情绪冰点）
- VIX波动率指数
- 融资余额/保证金比例
- 新发基金规模

**应对策略**:
- 情绪冰点：可逆势加仓
- 情绪过热：减仓锁定利润
- 恐慌踩踏：等待企稳信号

## 七、投研流程SOP

### 7.1 研究流程

```
宏观择时 → 行业比较 → 个券挖掘 → 深度研究 → 投资决策 → 跟踪复盘
   ↓           ↓           ↓           ↓           ↓           ↓
  ERP分位   景气度排序   初筛名单   财务建模   风控检查   归因分析
  股债配比   政策导向   实地调研   估值测算   仓位分配   策略迭代
```

### 7.2 日常监控清单

**每日必看**:
- [ ] 持仓标的价格/成交量异动
- [ ] 重大公告（业绩、订单、政策）
- [ ] 市场新闻（宏观、行业、公司）
- [ ] 北向资金/主力资金流向
- [ ] 海外市场（美股、汇率、商品）

**每周复盘**:
- [ ] 持仓标的基本面更新
- [ ] 行业景气度跟踪
- [ ] 估值水平变化
- [ ] 技术面关键点位
- [ ] 组合归因分析

**每月深度**:
- [ ] 宏观经济数据解读
- [ ] 行业比较更新
- [ ] 策略有效性评估
- [ ] 风险敞口分析
- [ ] 下月投资计划

### 7.3 投资决策流程

**买入决策**:
1. 基本面研究通过
2. 估值水平合理（有安全边际）
3. 催化剂明确（6个月内）
4. 风险评估可控
5. 仓位分配合理

**卖出决策**:
1. 基本面恶化（护城河受损）
2. 估值过高（泡沫化）
3. 止损线触发
4. 更好的替代标的
5. 宏观环境变化
