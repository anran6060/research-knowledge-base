## 1. 交易员的基本素质

**日期**：2026-03-03  
**来源**：公众号-岛上葫芦娃  
**关联研报**：-

### 核心观点
交易核心素质是反人性的心态和纪律系统：

1. **顺势而为** - 找顺着大方向的机会
2. **等待的能力** - 比交易更重要，像狙击手一样潜伏
3. **反人性操作** - 亏损时止损魄力，盈利时持仓定力
4. **心态70%+技术20%+运气10%**
5. **极简系统+持续总结**

## 2. 石油和有色价格上涨的成本传导分析框架

**日期**：2026-03-04  
**来源**：用户原创  
**关联研报**：高盛-亚洲股票地缘政治敏感性分析（化工板块受损验证）

### 核心观点
**成本依赖度排名**（油价/有色上涨时受影响最大）：
1. 化工
2. 装备制造业
3. 文体美工制品
4. 非金属矿物
5. 燃气供应

### 分析框架
```
成本指数 = Σ(A行业在40个工业行业的成本占比 × A行业PPI)

成本传导系数 = 成本指数涨幅 / 出厂价格涨幅
```

### 应用场景
- **识别利润挤压最严重行业**：成本涨 > 出厂价涨
- **识别定价权最强行业**：出厂价涨 > 成本涨
- 与高盛研报结论交叉验证：化工板块确实在受损名单中
