# 投资研究知识库 - MOC索引

本知识库用于系统化整理投研资料，支持语义搜索。

## 📁 目录结构

- [[01-日报]] - 每日市场复盘、宏观早报
- [[02-研报]] - 券商研报、行业分析
- [[03-观点]] - 投资观点、策略思考
- [[04-数据]] - 重要数据、图表

## 🏷️ 主题标签

### 资产类别
- #宏观 - 宏观经济、政策分析
- #债市 - 利率、信用、国债期货
- #权益 - 股票、板块分析
- #商品 - 大宗商品、能源、金属
- #外汇 - 汇率、国际资本流动

### 核心概念
- #利率周期
- #通胀预期
- #货币政策
- #财政政策
- #地缘政治

## 🔍 快速搜索

使用命令行搜索：
```bash
python3 ~/.knowledge_base/scripts/search_kb.py "你的搜索内容"
```

## 📝 添加文章

```bash
python3 ~/.knowledge_base/scripts/add_article.py \
  --title "文章标题" \
  --content "文章内容" \
  --source "公众号/研报" \
  --tags "宏观,债市"
```
